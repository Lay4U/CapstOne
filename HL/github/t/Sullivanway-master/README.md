# Sullivanway
일반적인 지하철앱을 구현하고, 가까운 엘리베이터 위치 정보, 휠체어 탑승자를 위한 환승지도 등 장애인을 위한 기능을 개발


## 사용 기술
- Android
> Firebase Cloud Messaging<br>
> 최소환승, 최단거리 구현<br>
> 특정 환승역 우회 알고리즘 개발
